# 投稿戦略マニュアル

投稿頻度・タイミング・フォーマット・コンテンツピラーの戦略基準です。
planner / director が参照します。
**具体的な設定値は `.claude/sns-profile.json` の `posting` セクションを参照してください。**

## 投稿頻度

- 1日の投稿数: `sns-profile.json` の `posting.frequency_per_day`（デフォルト: 3件）
- 月間上限: 1,500件（X API Free tier）
- 月末2週間は月間残数を確認し、ペースを調整する
- 残数が100件を切ったら director に報告し、今月の方針を確認する

## 投稿タイミング

`sns-profile.json` の `posting.best_times` に従う。デフォルト設定の根拠:

| 時間帯 | 理由 |
|--------|------|
| 7:00〜8:00 | 起床・通勤のスクロールタイム |
| 12:00〜13:00 | ランチタイムの隙間 |
| 19:00〜20:00 | 帰宅後のリラックスタイム |
| 22:00〜23:00 | 就寝前のタイム |

## フォーマット戦略

- **スレッド比率**: `posting.thread_ratio`（デフォルト: 30%）
  - 深い洞察、手順解説、体験談の続きに使う
  - 単発より拡散しにくいが、フォロワー化率が高い
- **画像付き比率**: `posting.image_ratio`（デフォルト: 20%）
  - 視覚的な情報整理、データグラフに使う

## コンテンツピラー

以下の比率を目安に組み合わせる:

| ピラー | 比率 | 目的 | 獲得しやすいエンゲージメント |
|--------|------|------|---------------------------|
| **教育系**（How-to / 知識） | 40% | 保存・引用RT獲得 | 保存、引用RT |
| **共感系**（体験談 / 感情） | 30% | いいね・リプライ獲得 | いいね、リプライ |
| **意見系**（主張 / 問いかけ） | 20% | 議論・RT獲得 | RT、リプライ |
| **告知系**（お知らせ / 誘導） | 10% | CV・誘導 | クリック |

告知系が10%を超えないよう注意する。多すぎると読者が離れる。

## トレンドジャック方針

- タイムリーなトレンドは積極的に活用する（analyst のリサーチを活用）
- ただし、自アカウントのジャンルと無関係なトレンドには乗らない
- 炎上中のトピックには触れない（`escalation-rules.md` 参照）
- トレンドジャックは週1〜2本まで。アカウントの軸がブレる原因になる

## 週次レビュー基準

evaluator の評価レポートで以下を確認し、次週の戦略に反映する:
- コンテンツピラー比率が目安通りか
- どのピラーのエンゲージメントが良かったか
- 投稿時間帯のパフォーマンス差
