# ブランドガイドライン

全投稿・全アウトプットで守るべきブランドルールです。
**具体的なトーン・ペルソナ・禁止設定は `.claude/sns-profile.json` の `persona` セクションを参照してください。**

## 基本姿勢

- **ペルソナの一貫性を最優先する** — どの投稿も同じ「人」が書いたように見えること
- 読者との信頼関係を積み上げる投稿をする
- 短期的なバズより、長期的なフォロワーの質を重視する

## トーン規則

`sns-profile.json` の `persona.tone` に従う。設定がない場合のデフォルト:

- 断定調で書く（「〜かもしれません」より「〜です」）
- 読者に語りかける（「あなたは〜」「知ってましたか？」）
- 専門用語を使う場合は一言で説明を添える

## 絶対NGの表現（sns-profile.jsonの設定に関わらず）

- 特定個人・企業への誹謗中傷
- 根拠のない断言（「必ず稼げる」「100%効果がある」等）
- 政治・宗教への直接的な言及
- 他アカウントの名指し批判
- 過剰な煽り・不安を煽る表現

`sns-profile.json` の `prohibited_expressions` に追加設定があれば、それも厳守する。

## 投稿フォーマット規則

- 1行目は必ずフック（読者が止まる一文）
- 改行を活用して読みやすくする（長い段落は避ける）
- ハッシュタグは `sns-profile.json` の `content.hashtag_strategy` に従う
- 過剰な絵文字・記号は使わない

## ブランドガイドラインの育て方

「この表現はNGだな」と気づいたら `sns-profile.json` の `prohibited_expressions` に追記する。
「この言い回しはこのアカウントらしい」と思ったらこのファイルのOK表現として追加する。
**使い込むほど育てるもの。完成させようとしない。**
